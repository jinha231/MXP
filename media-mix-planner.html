📦 매체 풀 관리
영상매체(VIDEO) 풀을 더 많이 추천받으려면 어떻게 해야 하나요? ▼
현재 영상(VIDEO) 그룹에 포함된 매체는 다음과 같습니다:
유튜브 유튜브 쇼츠 인스타그램 틱톡 페이스북 카카오TV 네이버 밴드 X(트위터) 네이버 GFA 카카오모먼트
추천 매체 수를 늘리는 3가지 방법:
  • ① 그룹 예산 비중 설정 — 광고설정 하단 "그룹별 예산 설정"에서 영상 VIDEO 비중(%)을 높이면 VIDEO 매체가 예산 배분 우선순위를 갖습니다.
  • ② 총 예산 증액 — 예산 규모에 따라 그룹당 최대 추천 매체 수가 자동 결정됩니다. (5천만 미만 → 2개, 1.5억 미만 → 3개, 5억 미만 → 4개, 5억 이상 → 5개)
  • ③ 결과에서 직접 추가 — 미디어믹스 결과 페이지 하단 ＋ 매체 추가 버튼으로 VIDEO 그룹 내 원하는 매체를 직접 추가할 수 있습니다.
💡 Tip 유튜브/인스타그램처럼 MEDIA_TIER 점수가 높은 매체(2.0, 1.8)는 동일 점수 대비 예산을 더 많이 배분받으므로 자동으로 우선 추천됩니다.
버티컬 매체 풀을 늘리고 싶어요 ▼
버티컬(VERTICAL) 그룹은 특정 라이프스타일·취미 앱 중심의 매체입니다:
토스 무신사 당근마켓 오늘의집 카카오페이 야놀자 여기어때 지그재그 화해 스노우 핑크다이어리 밀리의서재
  • ① 성별/연령 타겟 맞춤 — 버티컬 매체는 타겟 적합도에 따라 점수가 달라집니다. 예를 들어 20~30대 여성 타겟 설정 시 화해, 지그재그, 핑크다이어리 등이 자동 추천됩니다.
  • ② VERTICAL 예산 비중 설정 — 광고설정에서 버티컬 VERTICAL 비중을 지정하면 해당 그룹 매체가 먼저 배분됩니다.
  • ③ 관심사 태그 활용 — 광고설정 관심사에 "패션", "뷰티", "인테리어", "여행" 등을 입력하면 관련 버티컬 매체의 추천 점수가 높아집니다.
  • ④ 결과에서 직접 추가 — ＋ 매체 추가 버튼으로 원하는 버티컬 매체를 수동 추가하세요.
DSP 매체 풀을 확대하려면 어떻게 해야 하나요? ▼
DSP 그룹은 프로그래매틱 광고 플랫폼 중심입니다:
DV360(Google) The Trade Desk 크리테오 몰로코 Teads 코비(Covi) 앱피어(Appier) Cauly(카울리) 모비온 인크로스
  • ① DSP 예산 비중 설정 — 광고설정에서 DSP 비중을 지정합니다. DSP는 대규모 인벤토리 매입 방식이므로 총 예산 5천만원 이상에서 효율이 극대화됩니다.
  • ② 총 예산 증액 — 예산이 높을수록 그룹당 추천 매체 수가 늘어나 DSP 매체도 더 많이 포함됩니다.
  • ③ 퍼포먼스 관심사 태그 — 관심사에 "리타겟팅", "퍼포먼스", "앱설치" 등을 추가하면 DSP 계열 매체 점수가 높아집니다.
  • ④ 직접 추가 — 결과 페이지의 ＋ 매체 추가 버튼으로 원하는 DSP를 개별 추가하고 예산을 직접 입력하세요.
💡 DSP 선택 기준 DV360·TTD는 글로벌 인벤토리 커버리지, 크리테오는 리타겟팅, 몰로코는 앱 마케팅, Teads는 프리미엄 아티클 지면에 강점이 있습니다.
OTT 매체 풀도 늘리고 싶어요 ▼
OTT 그룹에 포함된 매체:
넷플릭스 티빙 쿠팡플레이 웨이브 왓챠 디즈니플러스
  • ① OTT 예산 비중 설정 — 광고설정에서 OTT 비중을 설정합니다.
  • ② 예산 규모 확인 — OTT 광고(특히 넷플릭스, 디즈니플러스)는 최소 집행 예산이 높습니다. 총 예산 1억원 이상에서 자동 추천이 활성화됩니다.
  • ③ 연령 타겟 조정 — OTT는 20~40대 타겟에 강하므로 해당 연령대를 선택하면 자동 추천 점수가 올라갑니다.
💰 예산 설정
그룹별 예산 비중은 어떻게 설정하나요? ▼
광고설정 페이지 하단의 "그룹별 예산 설정" 섹션에서 VIDEO / VERTICAL / OTT / DSP 각 그룹에 원하는 비중(%)을 입력합니다.
  • 입력 합계가 100%를 초과하면 경고가 표시됩니다.
  • 비중을 입력하지 않은 그룹은 점수 기반으로 자동 배분됩니다.
  • 예: VIDEO 40%, OTT 30% 입력 → 나머지 30%는 VERTICAL·DSP에 자동 분배
💡 추천 비중 (브랜드 캠페인 기준) 영상 40~50% · OTT 20~30% · 버티컬 15~20% · DSP 10~15%
VAT 포함/제외는 어떻게 다른가요? ▼
미디어믹스 결과 상단 "VAT 제외 / VAT 포함" 토글 버튼으로 전환할 수 있습니다.
  • VAT 제외 (기본) — 매체사 청구 금액 기준. 광고주가 대행사에 지급하는 순 광고비 기준으로 계획 시 사용합니다.
  • VAT 포함 — 부가세 10% 포함 금액. 예산서나 비용 승인 문서 작성 시 사용합니다.
💡 주의 내부 저장 데이터는 항상 VAT 제외 기준으로 관리되며, 화면 표시만 VAT 포함으로 전환됩니다. 엑셀 다운로드 시에도 현재 VAT 모드가 반영됩니다.
매체별 예산을 직접 조정할 수 있나요? ▼
네. 미디어믹스 결과 테이블의 예산 셀을 직접 클릭하면 수정할 수 있습니다.
  • 금액을 수정하면 예산 비중(%)이 자동으로 재계산됩니다.
  • 비중(%) 셀도 직접 수정 가능하며, 금액이 연동됩니다.
  • 하단 예산 상태 바에서 배분 합계, 잔여/초과 금액을 실시간으로 확인할 수 있습니다.
  • 잔여 예산을 자동으로 균등 분배하려면 ⚖ 자동 균형 버튼을 클릭하세요.
🎯 타겟팅 & 알고리즘
미디어믹스 추천 알고리즘은 어떻게 동작하나요? ▼
추천 알고리즘은 크게 3단계로 동작합니다:
  • ① 점수 산정 (scoreMedia) — 연령 매칭(+30), 성별 매칭(+20), 관심사 매칭(+20), 예산 적합도(+15), 고영향 매체 보너스(+10) 등을 합산합니다. 성별이 명확히 불일치하는 매체는 점수 0 처리(하드 필터)됩니다.
  • ② 그룹별 선발 (allocate) — 각 그룹에서 점수 상위 N개를 선발합니다. N은 예산 규모에 따라 2~5개로 자동 결정됩니다.
  • ③ 예산 배분 — 선발된 매체에 점수 × MEDIA_TIER 가중치 비율로 예산을 배분합니다. MEDIA_TIER가 높은 매체(유튜브 2.0, 인스타그램 1.8)가 더 많은 예산을 받습니다.
성별 타겟팅은 어떻게 작동하나요? ▼
광고설정에서 전체 / 남성 / 여성 중 하나를 선택합니다.
  • 성별이 "전체"이면 모든 매체가 추천 대상이 됩니다.
  • 남성 선택 시 — 화해, 지그재그, 핑크다이어리 등 여성 전용 매체는 점수 0으로 자동 제외됩니다.
  • 여성 선택 시 — 남성 타겟 비중이 높은 게임·스포츠 관련 매체 점수가 낮아집니다.
  • "전체" 매체로 분류된 플랫폼(유튜브, 카카오 등)은 성별에 무관하게 항상 추천 대상입니다.
광고 가이드·소개서 파일을 올리면 무엇이 달라지나요? ▼
  • 광고 가이드 (PDF) — 문서 내에서 매체명을 자동 인식해 해당 매체를 추천 풀에 우선 포함합니다. "임팩트", "SOV", "브랜드 인지도" 키워드가 있으면 High Impact 매체 추천 점수가 올라갑니다.
  • 소개서 (PDF) — 제품/서비스 카테고리를 파악해 관심사 태그와 연계합니다. 예: 뷰티 브랜드 소개서 → 화해, 지그재그, 인스타그램 점수 상승.
💡 Tip 파일 없이도 광고설정에서 관심사 태그를 직접 입력하면 동일한 효과를 얻을 수 있습니다.
📈 도달커브
도달률(Reach) 수치는 어떻게 해석하나요? ▼
도달률은 특정 예산으로 해당 매체의 타겟 모수(MAU) 중 몇 %에게 광고가 도달하는지를 시뮬레이션한 수치입니다.
  • R1 (1+ 도달) — 광고를 1회 이상 본 사람의 비율. 최대 도달 범위를 나타냅니다.
  • R3 (3+ 도달) — 광고를 3회 이상 본 사람의 비율. 광고 인지 효과가 실질적으로 발생하는 기준입니다.
💡 계산 모델 Erwin Ephron 네거티브 지수 모델: Reach = N × maxRate × (1 − e^(−imp/(N×freq))) 를 사용합니다. 광고비가 늘수록 도달률 증가폭이 점점 줄어드는 수확 체감 구조를 반영합니다.
매체 예산 비중을 바꾸면 도달률이 어떻게 달라지나요? ▼
도달커브 페이지에서 슬라이더 또는 % 입력칸으로 각 매체 예산 비중을 조정할 수 있습니다.
  • 동일 총 예산이라도 비중을 높인 매체는 더 높은 도달률을, 낮춘 매체는 낮은 도달률을 보입니다.
  • 예: 5억 예산을 유튜브 100%로 집행 시 유튜브 단독 R1 ~42% / 인스타그램 50%-유튜브 50% 시 각 R1 ~28%·~18%
  • 매체를 추가할수록 개별 도달률은 낮아지지만 총 순도달(unduplicated reach)은 증가하는 트레이드오프가 있습니다.
📁 파일 & 내보내기
엑셀로 다운로드하면 어떤 정보가 포함되나요? ▼
미디어믹스 결과 페이지의 📥 엑셀 다운로드 버튼을 클릭하면 다음 정보가 포함된 xlsx 파일이 생성됩니다:
  • 캠페인 기본 정보 (기간, 총 예산, 타겟)
  • 매체별 예산 배분 및 비중
  • 예상 노출수, 클릭수, CPM/CPC/CPV
  • 광고 기간 (개별 설정된 경우 매체별 날짜 포함)
  • Phase별 분리 시트 (Phase 설정이 있는 경우)
데이터가 자동 저장되나요? ▼
네. 모든 설정값과 미디어믹스 결과는 브라우저 localStorage에 자동 저장됩니다. 페이지를 새로고침하거나 브라우저를 닫아도 마지막 작업 상태가 복원됩니다.
  • 업로드한 소개서/가이드 파일 텍스트도 저장됩니다.
  • 로고 이미지(JPG/PNG)도 data URL 형태로 저장됩니다.
  • 저장 용량 초과 시 파일 데이터만 제외하고 나머지는 유지됩니다.
💡 초기화 설정 탭 → "데이터 초기화" 버튼으로 전체 저장 데이터를 삭제할 수 있습니다.
